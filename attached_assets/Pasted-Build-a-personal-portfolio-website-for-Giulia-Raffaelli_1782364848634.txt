Signature visual elements:
- Subtle CRT scanline overlay (repeating-linear-gradient, low opacity,
  mix-blend-mode overlay) + soft vignette — across the whole site, not just
  the hero
- A receding perspective grid floor (Tron-style) behind the hero only
- Pixel-style borders / chunky box-shadows on buttons and cards instead of
  soft drop shadows
- All hover/focus states have a visible neon outline (accessibility —
  visible keyboard focus is required)
- Mobile-first responsive; the pixel font is for short labels/titles only,
  never for paragraphs (readability)
